  • Please subscribe to the RPI Class of '79 electronic mailing list (RPI1979-L)
    To subscribe to this list, maintained by Joe Campbell, simply send an e-mail, from the address you wish to subscribe, to listproc@lists.rpi.edu and in the message body type:

    subscribe RPI1979-L First_Name (optional Maiden or Nickname) Last_Name

    For example: subscribe RPI1979-L Mary (Bayly) Skevington would subscribe "Mary (Bayly) Skevington" using the e-mail address she sent the request from.

    If you have a signature block on your e-mail message, turn it off or remove it from your subscription request. You should receive an automatic reply with information on using the list after you successfully subscribe. If you have any questions, please send a message to RPI1979-L-request@lists.rpi.edu.
